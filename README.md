# Real-Time Card Picker Tool

Python automation tool using computer vision and keyboard input to detect and select in-game targets with low latency.

Built as an experiment in screen capture, template matching, concurrency, and responsive desktop automation.

## Features

* Real-time screen capture
* Template matching with OpenCV
* Keyboard-triggered actions
* Audio confirmation feedback
* Configurable detection region and thresholds

## Tech Stack

* Python
* OpenCV
* NumPy
* MSS
* Pynput
* Pygame
